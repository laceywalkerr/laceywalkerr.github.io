# Lacey Walker Portfolio

Personal portfolio website for showcasing QA / SDET automation projects, technical writing, and professional experience.

## Live Site

https://laceywalkerr.github.io

## Features

- Professional QA / SDET portfolio
- Automation framework project showcases
- Technical blog ("Field Notes")
- Resume and experience overview
- Contact links

- ## Built With

- HTML5
- CSS3
- Responsive layout (CSS Grid / Flexbox)
- Hosted with GitHub Pages

- ## Running Locally

Clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/laceywalkerr/laceywalkerr.github.io.git
```
